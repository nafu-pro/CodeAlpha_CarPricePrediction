# Car Price Prediction Using Machine Learning

## Project Overview
This project predicts the selling price of cars using Machine Learning. The model is trained using various car features such as year, present price, kilometers driven, fuel type, seller type, transmission type, and owner information.

## Dataset Features
- Car_Name
- Year
- Present_Price
- Driven_kms
- Fuel_Type
- Selling_type
- Transmission
- Owner

## Target Variable
- Selling_Price

## Technologies Used
- Python
- Pandas
- Scikit-learn
- Machine Learning
- Linear Regression

## Model Performance
- R² Score: 0.8465
- MAE: 1.2147

## Project Workflow
1. Data Loading
2. Data Preprocessing
3. Label Encoding
4. Feature Selection
5. Train-Test Split
6. Linear Regression Model Training
7. Prediction and Evaluation

## Results
The Linear Regression model achieved good prediction performance with an R² Score of 84.65%.
